### Описание 

В ходе выполнения лабораторной работы были изучены базовые основы языка программирования JavaScript и проведено сравнение его концепций с языком C#.

### Ключевые темы:

1. ***Подготовка окружения***
   - Настройка рабочего пространства в Visual Studio Code
   - Инициализация Git-репозитория и работа с GitHub

2. ***Подключение JavaScript к HTML***
   - Базовая структура HTML-документа
   - Подключение внешних скриптов через тег `<script>`

3. ***Циклы в JavaScript***
   - Циклы for, while, do...while
   - Операторы управления циклом: break и continue
   - Вложенные циклы

4. ***Функции***
   - Объявление и вызов функций
   - Параметры по умолчанию
   - Стрелочные функции (Arrow Functions)
   - Функции как значения (аналог делегатов в C#)

5. ***Массивы***
   - Создание и доступ к элементам
   - Методы добавления/удаления: push(), pop()
   - Перебор массивов: for, for...of
   - Поиск элементов: indexOf(), includes()

6. ***Объекты***
   - Создание объектов через литералы (object literal)
   - Доступ к свойствам через точку и квадратные скобки
   - Методы объектов и ключевое слово this
   - Перебор свойств через for...in

### Основные отличия JavaScript от C#:
| Характеристика | C# | JavaScript |
|----------------|-----|------------|
| Типизация | Статическая | Динамическая |
| Объявление функций | Методы класса | Function / Arrow |
| Массивы | Фиксированный тип | Могут содержать разные типы |
| Объекты | Требуют класс | Создаются без класса |
